# goexec

A single liner to execute source code programs.

## Installation

```bash
go get -u github.com/goexec/goexec
```

## Usage

```bash
goexec [flags] [program] [arguments]
```

## Flags

```bash
-h, --help
    Prints help information.
-v, --version
    Prints version information.
```

## Examples

```bash
goexec -v
goexec -h
goexec ./main.go
goexec ./main.go -h
goexec ./main.go -v
```
